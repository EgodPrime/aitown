# 故事索引

此目录包含从 PRD 分片中提取并整理的开发者友好用户故事草案。每个文件对应 `docs/prd/` 下的同编号分片，便于评审与拆分为 sprint 任务。

- `01-overview-stories.md` - 项目高层目标与演示性故事
- `02-scope-stories.md` - 范围与约束故事
- `03-stories-stories.md` - PRD 中已列示故事的细化
- `04-mvp-features-stories.md` - MVP 核心功能的故事与验收
- `05-frontend-stories.md` - 前端展示与交互故事
- `06-memory-stories.md` - 记忆与日志相关故事
- `07-llm-adapter-stories.md` - LLM 适配器与降级策略故事
- `08-flows-stories.md` - 关键交互流程故事
- `09-api-stories.md` - API 契约与测试故事
- `10-acceptance-stories.md` - 可执行的验收/测试故事
- `11-nonfunctional-stories.md` - 非功能需求与实施故事

下一步建议：将每个故事拆解为具体任务（Jira/Github Issue），并为高优先级故事编写技术任务与估时。
